Frequently Asked Questions

How does it work in the browser?

Rust Video Editor uses WebAssembly to run high-performance Rust code directly in your browser, providing native-like performance without downloads.

What video formats are supported?

We support all major formats including MP4, WebM, MOV, AVI, and more. The editor automatically handles format conversion as needed.

Is my data secure?

Yes! All processing happens locally in your browser. Your videos are only uploaded to our servers if you choose to use cloud storage.

Can I use it offline?

Yes, once loaded, the editor works offline. You can edit videos without an internet connection and sync when you're back online.

What browsers are supported?

Rust Video Editor works on all modern browsers including Chrome, Firefox, Safari, and Edge. We recommend using the latest version for best performance.

Can I collaborate with others?

Team plans include real-time collaboration features, allowing multiple users to work on the same project simultaneously.
